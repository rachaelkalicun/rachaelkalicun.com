## Don't Break the Chain

It's a new year (and now even an [Old New Year](https://en.wikipedia.org/wiki/Old_New_Year){:target="_blank"}) and an opportunity for a new approach to keep Rachael 2.0 moving in the right direction. 

**The Plan:**
Use the Github contribution graph on [my Github profile](https://github.com/rachaelkalicun){:target="_blank"} to encourage never breaking the chain (weekdays for now). 

<div>
	<img src="/images/sprints/github-dont-break-the-chain.jpg" alt="github don't break the chain graph"/>
</div>

**How?** Each day, push a commit only after all five focus areas have been attended to at least a little bit. Each day consists of only one ticket on the sprint board. Create a daily summary to briefly explain how I touched each category. 

**Why?** This new approach simplifies, allows flexibility and reduces overhead while building consistency with the reward of getting the box filled in!  

This week, I aim to work on five focus areas every day:
